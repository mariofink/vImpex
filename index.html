vImpex

Stupid name for a little tool to visualise the content of impex files used by the Hybris ecommerce platform.

Instructions:

Just drag an impex file from your desktop onto this window to have its contents displayed in a more readable tabular format.
If it only helps you to keep away from opening Excel occasionally then that’s a good thing, right?

Code is on github

There are still some bugs and glitches, so have fun fixing them. If that’s not your thing, you can also file an issue.
